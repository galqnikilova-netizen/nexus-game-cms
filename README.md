# NEXUS Game CMS

Modern self-hosted CMS for gaming communities, built on Laravel 12.

> Status: active development. Do not deploy to production yet.

## First milestone

- cPanel and CyberPanel installer
- Clean URL routing
- Authentication and role-based admin panel
- Responsive NEXUS public theme
- Game server management and public monitoring
- Bulgarian and English localization
- Full install, update and rollback packages

The previous experimental PHP alpha is not used in this codebase.
